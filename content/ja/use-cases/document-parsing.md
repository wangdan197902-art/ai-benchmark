# Document Parsing

## ユースケース概要

This page focuses on the Document Parsing use case, covering core requirements, recommended models, and evaluation criteria.

## 選択基準

- 上下文窗口 ≥ 32K
- 支持图像/音频输入

## 評価基準

- 上下文窗口 ≥ 32K
- 支持图像/音频输入

## ベストプラクティス

- 提供系统提示以建立上下文
- 监控 token 用量与成本
