# 代码审查

## 场景概述

本页聚焦 代码审查 场景，介绍该场景的核心需求、推荐模型以及评估标准。

## 选型标准

- HumanEval 得分 ≥ 70
- 上下文窗口 ≥ 32K

## 评估标准

- HumanEval 得分 ≥ 70
- 上下文窗口 ≥ 32K

## 最佳实践

- 提供系统提示以建立上下文
- 监控 token 用量与成本
